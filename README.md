
# まさもん

#### このアプリでは何ができるのか？
* シフト表のエクセルファイルやPDFファイルを取り込んで、月給の計算やシフトを1日単位で簡単に見ることができる

#### バージョンとか
* xcode 7.2.1
* iOS 9.2.1
* iPhone 6,6s

#### 主な機能
* 時給登録
* 給与計算
* シフトの取り込み

#### 使用したライブラリ
* XlsxReaderWriter
* SwipeBetweenViewControllers
* handMadeCalendarOfSwift
* Gradient Circular Progress
* TET
